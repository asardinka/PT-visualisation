# Phase Portrait

Демонстрационное приложение для визуализации фазовых портретов 
системы двух дифференциальных уравнений вида dx/dt = f(x,y), dy/dt = g(x,y).

## Возможности

- Ввод произвольных функций f(x, y) и g(x, y)
- Автоматический поиск и классификация особых точек (седло, фокус, центр, узел)
- Отображение векторного поля
- Анимация траектории из заданной начальной точки (x0, y0)
- Zoom колёсиком мыши, перемещение холста мышью

## Структура проекта

```
Phase_project/
├── main.cpp          # Главный цикл: связывает все модули
├── CMakeLists.txt    # Сборка
├── run.ps1           # Скрипт сборки и запуска
└── src/
    ├── ode.hpp       # Парсинг f и g через exprtk, шаг интегрирования RK4
    ├── analyzer.hpp  # Поиск особых точек, якобиан, классификация (Eigen)
    ├── canvas.hpp    # Рендер: сетка, векторное поле, траектории, zoom/pan
    └── ui.hpp        # ImGui панель: поля ввода, кнопки, слайдеры
```

## Зависимости


| Библиотека | Назначение                       | Подключение     |
| ---------- | -------------------------------- | --------------- |
| SFML 3.0   | Окно                             | libs/SFML       |
| ImGui      | GUI панель                       | libs/imgui      |
| imgui-sfml | Интеграция ImGui с SFML          | libs/imgui-sfml |
| Eigen      | Собственные значения матрицы     | libs/Eigen      |
| exprtk     | Парсинг математических выражений | libs/exprtk.hpp |


## Требования

- Windows
- MSYS2 UCRT64: g++ 15+
- CMake 4.0+

## Запуск

```
.\run.ps1
```

При первом запуске сборка занимает 2-3 минуты из-за компиляции exprtk.
Последующие запуски значительно быстрее.
