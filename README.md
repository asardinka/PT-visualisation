# Phase Portrait

Визуализация фазовых портретов плоской системы **dx/dt = f(x,y), dy/dt = g(x,y)**.

## Готовая сборка

Можно использовать уже скомпилированный `**PhasePortrait.exe`** (Windows **x64**). В приложении реализован функционал:

- Ввода **f** и **g** как выражений; в формулах используются **x**, y, t.
- Поиск особых точек, отображение их на плоскости и классификация типа (седло, узел, фокус, центр и т. п.).
- Векторное поле и траектория, начинающаяся в заданной точке **(x0, y0)**.
- Точка движется вдоль построенной траектории на фазовой плоскости.
- Колёсико мыши меняет масштаб, перетаскивание сдвигает видимую область.

На фазовой плоскости маркеры особых точек: **зелёный** — устойчивая точка, **красный** — неустойчивая.

## Сборка из исходников

### 1. Папка `libs/`

Создайте каталог `**libs`** в корне проекта и поместите в него зависимости (пути должны совпадать с ожидаемыми в `CMakeLists.txt`):


| Что положить                                                      | Куда именно                                                  |
| ----------------------------------------------------------------- | ------------------------------------------------------------ |
| **SFML 3** (Graphics, Window, System), префикс с `lib/cmake/SFML` | `libs/SFML/` (чтобы существовал `libs/SFML/lib/cmake/SFML/`) |
| **Dear ImGui** (исходники)                                        | `libs/imgui/`                                                |
| **imgui-sfml** (в т.ч. `imgui-SFML.cpp`)                          | `libs/imgui-sfml/`                                           |
| **Eigen** (заголовочная библиотека)                               | `libs/Eigen/`                                                |
| **exprtk** (один заголовок)                                       | `libs/exprtk.hpp`                                            |


Версия компилятора: **MinGW-w64 UCRT64** (например из MSYS2), **CMake 3.16+**.

### 2. Скрипт `run.ps1` (Windows, PowerShell)


| Команда                       | Действие                                                                         |
| ----------------------------- | -------------------------------------------------------------------------------- |
| `.\run.ps1`                   | Сборка в `**build/Debug`** (динамический SFML), копирование `sfml-*.dll`, запуск |
| `.\run.ps1 -Configure`        | То же, но заново вызывает CMake                                                  |
| `.\run.ps1 -RunOnly`          | Только запуск уже собранного `**build/Debug\PhasePortrait.exe`**                 |
| `.\run.ps1 -Release`          | Сборка в `**build/Release`**, статический SFML, готовый exe для переноса (x64)   |
| `.\run.ps1 -Release -Run`     | После релиз-сборки запустить exe                                                 |
| `.\run.ps1 -Release -RunOnly` | Только запуск `**build/Release\PhasePortrait.exe`**                              |


Первый полный проход компиляции `**odeSystem.cpp**` (exprtk) может занять несколько минут; дальше — инкрементально.

### 3. Структура исходников

```
main.cpp, CMakeLists.txt, run.ps1
src/
  odeSystem.cpp / odeSystem.hpp  — разбор формул f и g, построение траекторий, определение особых точек
  canvas.hpp                     — сетка, поле, траектория, масштаб и сдвиг холста
  ui.hpp                         — ImGui-панель
```
